# Research Rewards

A Factorio 2.1 mod that grants players a sample stack of any newly craftable items that are unlocked whenever a research completes.

## Mod compatibility

- **Vanilla** — barrels (`[fluid]-barrel`) are used as fluid containers when the "Grant liquid containers" setting is enabled.
- **Angels Petrochem / Refining** — gas canisters and liquid containers are supported. Angels uses the same name for a fluid and its container item (e.g. `angels-gas-hydrogen`), so the correct item is granted automatically.

## Requirements

- Factorio 2.1+
